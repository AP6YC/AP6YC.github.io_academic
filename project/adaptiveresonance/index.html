    AdaptiveResonance.jl

    Sep 27, 2021
    Post GitHub JuliaHub Publication
    ART module architecture.

    AdaptiveResonance.jl is a registered JuliaHub package for Adaptive Resonance Theory (ART) modules. These algorithms are built upon a foundation of the neural theory and implementations of Stephen Grossberg and Gail Carpenter.

    The contributions of the project are represented as an accepted paper in the Journal of Open Source Software (JOSS).
